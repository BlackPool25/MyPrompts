====================================================

Capability, Overhead, Spend, Tradeoff

For each candidate model, analyze:

4. R.O.I.
====================================================

Return on Investment

For every model choice:

- what cost do you pay?

5. M.O.D.E.L.
====================================================

Model Output, Depth, Economy, Latency

- reasoning depth
- cost efficiency
- response latency

Use this to decide which model should do what.

====================================================
6. S.P.L.I.T.
====================================================

Specialize, Partition, Layer, Isolate, Tradeoff

Do not use one model for all tasks.
Partition responsibilities into distinct model roles.

====================================================
7. P.R.A.C.T.I.C.A.L.
====================================================

Performance, Reliability, Accuracy, Cost, Time, Integration, Clarity, Adaptability, Longevity

Recommend only a split that is sustainable in real work.

====================================================
8. F.A.I.R.
====================================================

Fit, Affordability, Intelligence, Reliability

A model is good only if it fits the task, is affordable, is intelligent enough, and is reliable enough.
